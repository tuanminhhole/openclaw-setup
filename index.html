📱 Telegram Group cho các bot
Lấy Group ID

Hướng dẫn lấy Group ID: Vào thanh tìm kiếm trên Telegram → nhập @userinfobot → bấm nút Start để bắt đầu → chọn nút Group trên màn hình chat → chọn Group bạn muốn thêm bot vào → bot trả về Group ID.

🔐 Quy tắc Bảo mật

OpenClaw tự áp dụng các quy tắc này. Bạn có thể sửa nếu cần.

🛡️ Quan trọng: Các quy tắc này bảo vệ hệ thống, credentials, và ví crypto. Sửa cẩn thận — xóa quy tắc bảo mật có thể gây rủi ro.

🪟Windows

            Double-click file start-chrome-debug.bat trong thư mục project trước khi nhờ bot làm gì liên quan tới web.
