Cấu hình Bot —

Chọn AI model, đặt tên, tính cách và quy tắc cho Bot của bạn.

🤖 Danh tính & Tính cách Bot

Các thông tin trên sẽ vào IDENTITY.md — nơi bot biết tên mình và cách xưng hô.

📝 Nội dung trên sẽ vào SOUL.md (tính cách) và AGENTS.md (quy tắc vận hành). OpenClaw tự động inject vào context đầu mỗi session.
👤 Nội dung trên sẽ vào USER.md — bot dùng để cá nhân hóa cách trả lời cho bạn. Để trống nếu chưa muốn điền.
🔐 Quy tắc Bảo mật

OpenClaw tự áp dụng các quy tắc này. Bạn có thể sửa nếu cần.

🛡️ Quan trọng: Các quy tắc này bảo vệ hệ thống, credentials, và ví crypto. Sửa cẩn thận — xóa quy tắc bảo mật có thể gây rủi ro.

Thêm kênh chat hoặc tính năng runtime. Cài qua npm (openclaw plugins install).

🔀 Sau khi Docker chạy — Setup 9Router

        

          

            1
            Mở trình duyệt → localhost:20128/dashboard
          

          

            2
            Đăng nhập OAuth vào các AI provider bạn muốn dùng (Google, Claude, Qwen...)
          

          

            3
            9Router tự động route request từ OpenClaw → AI provider tối ưu nhất. Done! 🎉
          

        

        

          ⚠️ Lưu ý: 9Router là tool cộng đồng (open-source). Kiểm tra Terms of Service của từng AI provider trước khi dùng.
        

      


      
      

        
🌐 Browser Automation — Bật Chrome Debug Mode

        

          Bot cần kết nối tới Chrome trên máy bạn qua Debug Mode. Lưu file .bat bên dưới và chạy trước khi khởi động Docker.
        


        
        

          

            start-chrome-debug.bat
            
          

          

        


        
        

          

            ⏰ Tự động bật Chrome Debug khi mở máy: Chạy script PowerShell bên dưới 1 lần (Run as Administrator) để đăng ký Task Scheduler:
          

          

            

              setup-chrome-task.ps1
              
            

            

          

        


        

          💡 Lưu ý:
          
    
            
  • Chrome debug dùng profile riêng — bạn cần login lại Facebook/Gmail 1 lần
    • 
            
  • Phải bật Chrome debug trước khi Docker chạy (hoặc restart Docker sau)
    • 
            
  • Port 9222 chỉ bind localhost — an toàn trên mạng nội bộ
    • 
            
  • Nói với bot "dùng browser mở..." để bot chủ động dùng trình duyệt
    • 
          

        

      


      
      

        
🔧 Thêm / bỏ Skills & Plugins sau khi setup

        

          Không cần chạy lại wizard. Dùng lệnh bên dưới trực tiếp trong Docker container:
        

        

          

            Terminal
            
          

          
# Vào container
docker exec -it openclaw-bot sh

# Thêm skill mới
openclaw skills install web-search
openclaw skills install memory

# Xóa skill
openclaw skills uninstall web-search

# Thêm plugin mới
openclaw plugins install @openclaw/voice-call

# Xóa plugin
openclaw plugins uninstall @openclaw/voice-call

# Restart để áp dụng
exit
docker restart openclaw-bot

        

        

          💡 Tip: Nếu muốn thay đổi lâu dài (persist qua Docker rebuild), hãy sửa Dockerfile rồi docker compose build && docker compose up -d.
