Cấu hình Bot —

Tên bot sẽ hiển thị trong system prompt

🛡️ Lưu ý: System Prompt chỉ định vai trò & tính cách bot. Các quy tắc bảo mật hệ thống (không xóa file, không truy cập thư mục nhạy cảm, không lộ API key...) được OpenClaw tự động áp dụng — bạn không cần viết vào đây.

⚠️ Zalo Personal — Cài trực tiếp trên Windows

          

            Zalo Personal dùng thư viện zca-js — hiện chỉ chạy trên Windows (không hỗ trợ Docker).
          

        


        
        

          
📦 Bước 1: Cài Node.js

          

            Nếu chưa có → tải tại nodejs.org → cài xong qua Bước 2.
          

        


        
        

          
🚀 Bước 2: Mở PowerShell → Paste → Enter

          

            ⚠️ Nhớ thay API key ở dòng đầu bằng key thật (lấy ở bước trước).
          

          

            

              PowerShell — Paste nguyên khối
